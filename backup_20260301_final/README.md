# 🌟 みんなのアンケート広場 (minna-no-vote-square) 🌟

誰でもかんたんに、匿名でも Google ログインでもアンケートを作成・投票できる、みんなの「気になる」が集まる場所です！🚀

## 🔗 サイト URL
[https://minna-no-vote-square.vercel.app/](https://minna-no-vote-square.vercel.app/)

## ✨ 主な機能
- **🗳️ かんたん投票**: 選択肢をクリックするだけで、誰でもすぐに意見を共有。
- **📝 アンケート作成**: お題、カテゴリ、締切を決めて、自分だけのアンケートを公開。
- **💬 盛り上がる掲示板**: 投票だけじゃない！コメント欄でリアルタイムに意見交換。
- **🔍 探しやすさ**: 新着、人気、カテゴリ別、ウォッチリスト機能で気になるお題を逃さない。
- **🔐 安心の公開設定**: 「全体公開」「限定公開（URLを知っている人のみ）」「非公開」を使い分け。
- **🎨 究極の UI**: モバイルファーストで設計された、美しく使いやすいデザイン。

## 🛠️ 技術スタック
- **Frontend**: React 19 / Vite
- **Database / Auth**: Supabase (PostgreSQL)
- **Styling**: Vanilla CSS (Premium Custom Design)
- **Notification**: EmailJS
- **Ad**: Google AdSense (審査中)

## 📦 開発・実行方法
1. リポジトリをクローン
2. `npm install` で依存関係をインストール
3. `.env` に Supabase の環境変数を設定
4. `npm run dev` でローカル開発サーバーを起動
